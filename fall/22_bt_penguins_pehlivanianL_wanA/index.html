Quick Bootstrap Tutorial

What is the purpose of Bootstrap?

  • Faster and easier web development.
  • Includes design templates for website elements like buttons, forms, etc.
  • Responsive Web Design: creating websites that automatically adjust themselves to look good on differently-sized devices (desktop, mobile, etc.)

Get started with Bootstrap: A helpful link with Bootstrap CSS classes.

Link Bootstrap stylesheet

Apply built-in Bootstrap formatting

The Bootstrap stylesheet contains many built-in classes and css for those classes.

There are built-in classes for typography, buttons, forms, helpers, images, tables, dropdowns, and navigation components.

For example:

    ▪.btn-primary creates a blue button.

    ▪.img-rounded makes the edges of an image rounded.

    ▪.text-mutedmakes secondary faded text next to a heading (we included an example above!)

    You can use the built-in formatting by setting the HTML class: <tag class="class_name">

    ▪ Mark Tags Highlights the text in yellow. ex: <mark>Omelette</mark> creates Omelette

You can see a list of all of the classes in the link above.

Responsive elements

If you zoom in/out on this webpage, the text automically shifts for the new size. (If you zoom out enough the components should merge into one table). This is an example of responsive web design.

The Grid System is a way to create responsive webpages.

You can create a grid by first creating a div with class="container".
    To make the grid responsive, set the container class to container-fluid, which sets the width to 100% across any sized window.

To add a new row to the grid, create a new div inside with class="row"

Then, within each row, you can create up to 12 columns. Grid columns are created by specificing the size of the window and then the number of the 12 available columns you wish to span.
  ▪For example, setting class="col-sm-4" sizes your grid for a small window and creates a column that takes up 4 blocks.
   Having 3 columns size 4 would take up the entire screen because there are only 12 available columns per row.
  ▪The available sizes are xs, sm, md, and lg.
  ▪To make the columns responsive, set the class to col-*-*. This allows the size of the column and window to adjust.

Example grid code: The code below makes two equally-sized columns in a fluid container.


              <div class="container-fluid">
                <div class="row">
                  <div class="col-*-*">
                    <h1>Good morning</h1>
                  </div>
                  <div class="col-*-*">
                    <h1>Good night</h1>
                  </div>
                </div>
              </div>
